Dashboards
==========

The dashboards page shows graphs and meters pulled from data in Grafana in kiosk mode. You can access the Grafana instance backing this dashboard `here </grafana/>`_. See the `Grafana documentation <https://grafana.com/docs/grafana/latest/getting-started/getting-started/>`_ for instructions on how to add and configure this dashboard.
